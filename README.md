# Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript.

## Features

- Two-player gameplay
- Interactive game board
- Winner detection
- Draw detection
- Restart/New Game option
- Responsive and user-friendly interface

## How to Play

1. Open the game in your browser.
2. Player X starts first.
3. Players take turns clicking on empty cells.
4. The first player to align three marks horizontally, vertically, or diagonally wins.
5. If all cells are filled and no player wins, the game ends in a draw.
6. Click the **Restart** or **New Game** button to play again.
